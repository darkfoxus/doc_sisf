USER | UPDATE

Permite modificar los datos de un usuario

Ver API Ver en GitHub

Update User

Url: http://sisf.herokuapp.com/user/update

Metodo: POST

Recibe:

  • {token, id_user, new_password, new_rol, new_name, new_identification} | Si se es Administrador y se editara un usuario
  • {token, new_name, new_identification, new_password} | Para modificar usuario actual

Responde:

  • HTTP 200 | Si el user fue editado
  • HTTP 403 | Permisos insuficientes
  • HTTP 405 | Error
